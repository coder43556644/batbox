COMMANDE BATBOX

     BATBOX est une commande externe qui permet la CRéATION de GRAPHISMES 
     directement dans la console, mais aussi de gérer des interactions avec 
     l'utilisateur.

     Par exemple, BATBOX peut déplacer le curseur dans la console, changer 
     localement la couleur de celle-ci, ou encore récupérer les clics de 
     l'utilisateur dans la console.

SYNOPSIS

         BATBOX [/d Texte] [/g X Y] [/c Couleur] [/m] [/k[_]] [/a Caractere] 
      [/w Temps] [/f Etat] [/s Fichier] [/o OffsetX OffsetY] [/h mode] [/p 
      mode] [/n] [/y]...
         BATBOX [/disp text] [/goto X posY] [/color col] [/mouse] [/key[_]] 
      ...
         :: Anciens noms de fonctions deprecies (version 1.0)

     Permet de manipuler l'apparence de la console:

        - /D : Affiche du texte. Voir BatBox (/d) (-> ``_batbox/disp'') .

        - /G : Déplace le curseur. Voir BatBox (/g) (-> ``_batbox/goto'') .

        - /C : Change couleur de la console. Voir BatBox (/c) (-> 
          ``_batbox/color'') .

        - /M : Récupère un clic. Voir BatBox (/m) (-> ``_batbox/mouse'') .

        - /Y : Récupère un événement de la souris. Cette commande est 
          uniqument disponnible depuis la version 3.0 . BatBox (/y) (-> 
          ``_batbox/event'') 

        - /K[_] : Récupère une touche. Voir BatBox (/k) (-> ``_batbox/key'') .

        - /A : Affiche le caractère d'un code ASCII. Depuis la version 2.4, 
          BATBOX supporte aussi les caractères des encodages multi-octets 
          (principalement UTF-8). Voir BatBox (/a) (-> ``_batbox/ascii'') .

        - /W : Effectue une temporisation. Voir BatBox (/w) (-> 
          ``_batbox/wait'') .

        - /F : Change le mode de la console. Voir BatBox (/f) (-> 
          ``_batbox/fullscr'') . Cette option ne marche que sous WINDOWS XP. 
          En effet, les pilotes vidéo de la console n'offrent plus de mode 
          plein écran depuis WINDOWS VISTA. Cette commmande n'est plus 
          disponible depuis la version 3.0.

        - /S : Joue une musique à partir d'un fichier WAV. Voir BatBox (/s) 
          (-> ``_batbox/sound'') . Cette option n'est disponible en tant que 
          telle uniquement depuis la version 2.4. Avant la version 2.3, elle 
          permettait de jouer une note, mais n'était ni simple a utiliser, ni 
          robuste. Pour l'ancienne commande, Voir BatBox (/s)(Ancien) (-> 
          ``_batbox/sound_old'') . Depuis la version 3.0, cette commande n'est 
          plus disponible.

        - /O : Déplace l'origne de la console. Cette commande est très utile 
          pour la création de sprites via batbox. Voir BatBox (/o) (-> 
          ``_batbox/offset'') .

        - /H : Affiche ou cache le curseur de la console. Cette commande est 
          disponible depuis la version 2.4. Voir BatBox (/h) (-> 
          ``_batbox/cursor'') .

        - /P : Change le mode de la console . Cette commande est disponible 
          depuis la version 2.4. Voir BatBox (/p) (-> ``_batbox/console'') .

        - /N : Retourne à la ligne, tout en restant en dessous des dernières 
          coordonées ou un GOTO a eu lieu. Disponible depuis la version 3.0 . 
          Voir BatBox (/n) (-> ``_batbox/newline'') 

     Si vous voulez créer des graphismes en utilisant BATBOX, il est possible 
     de générer une ligne de commande pour BATBOX à partir d'un fichier texte 
     contenant une image en ASCII-ART, en utilisant l'utilitaire SpriteBox (-> 
     ``spritebox'') .

NOTES

     On peut combiner plusieurs fonctions sur une seule ligne de la commande 
     BATBOX, comme le signalent les ....

     Ainsi, le script suivant:

         BATBOX /g 10 10
         BATBOX /d "Hello world"
         BATBOX /g 0 0

     Peut être raccourci en :

         BATBOX /g 10 10 /d "Hello world" /g 0 0

     Il est possible de spécifier les nombres dans BATBOX dans trois bases 
     différentes, et cela pour toutes les options. La première est la base 
     DéCIMALE, pour spécifier cette base, il suffit de mettre le nombre en 
     décimal. La seconde est la base HEXADéCIMALE, particulièrement utilisée 
     pour les couleurs. Pour spécifier un nombre dans la base HEXADéCIMALE, il 
     faut ajouter à ce nombre le préfixe 0X. La dernière est la base OCTALE, 
     dont l'usage est déconseillé car cette base est peu usitée. Pour 
     spécifier un nombre en base OCTALE, il faut ajouter à ce nombre le 
     préfixe 0.

     n'est disponible que sous MS-WINDOWS.
