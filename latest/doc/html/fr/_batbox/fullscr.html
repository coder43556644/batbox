BatBox /f

Permet de mettre la console en mode plein écran ou en mode fenêtre.

Synopsis

BATBOX /f mode

Change le mode de la console.

  • mode : mode à appliquer à la console. Doit être une des deux valeurs suivantes

    • 1 : mode plein écran

    • 2 : mode fenêtre

Cette commande n'est plus disponible depuis la version 3.0

Note

Le mode plein écran ne semble être supporté que sous Windows XP.
