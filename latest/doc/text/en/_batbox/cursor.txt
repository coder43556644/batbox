BATBOX (/H)

     Shows or hides command prompt's cursor.

SYNOPSIS

         BatBox /h mode

     Shows or hides command prompt's cursor.

        - MODE : Specify the cursor mode to be set. It can be one of the 
          following value (by default 1):

             - 0 : The cursor is hidden.

             - 1 : The cursor is shown.

A VOIR AUSSI

     BatBox (/a) (-> ``ascii'') , BatBox (/m) (-> ``mouse'') , BatBox (/f) (-> 
     ``fullscr'') , BatBox (-> ``../batbox'')
