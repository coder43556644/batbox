Commande BatBox

BatBox est une commande externe qui permet la création de graphismes directement dans la console, mais aussi de gérer des interactions avec l'utilisateur.

Par exemple, BatBox peut déplacer le curseur dans la console, changer localement la couleur de celle-ci, ou encore récupérer les clics de l'utilisateur dans la console.

Synopsis

BATBOX [/d Texte] [/g X Y] [/c Couleur] [/m] [/k[_]] [/a Caractere] [/w Temps] [/f Etat] [/s Fichier] [/o OffsetX OffsetY] [/h mode] [/p mode] [/n] [/y] [/x sprite]...
BATBOX [/disp text] [/goto X posY] [/color col] [/mouse] [/key[_]] ...
:: Anciens noms de fonctions deprecies (version 1.0)

Permet de manipuler l'apparence de la console:

  • /d : Affiche du texte. Voir BatBox (/d).

  • /g : Déplace le curseur. Voir BatBox (/g).

  • /c : Change couleur de la console. Voir BatBox (/c).

  • /m : Récupère un clic. Voir BatBox (/m).

  • /y : Récupère un événement de la souris. Cette commande est uniqument disponnible depuis la version 3.0 . BatBox (/y)

  • /k[_] : Récupère une touche. Voir BatBox (/k).

  • /a : Affiche le caractère d'un code ASCII. Depuis la version 2.4, BatBox supporte aussi les caractères des encodages multi-octets (principalement UTF-8). Voir BatBox (/a).

  • /w : Effectue une temporisation. Voir BatBox (/w).

  • /f : Change le mode de la console. Voir BatBox (/f). Cette option ne marche que sous windows XP. En effet, les pilotes vidéo de la console n'offrent plus de mode plein écran depuis Windows Vista. Cette commmande n'est plus disponible depuis la version 3.0.

  • /s : Joue une musique à partir d'un fichier WAV. Voir BatBox (/s). Cette option n'est disponible en tant que telle uniquement depuis la version 2.4. Avant la version 2.3, elle permettait de jouer une note, mais n'était ni simple a utiliser, ni robuste. Pour l'ancienne commande, Voir BatBox (/s)(Ancien). Depuis la version 3.0, cette commande n'est plus disponible.

  • /o : Déplace l'origne de la console. Cette commande est très utile pour la création de sprites via batbox. Voir BatBox (/o).

  • /h : Affiche ou cache le curseur de la console. Cette commande est disponible depuis la version 2.4. Voir BatBox (/h).

  • /p : Change le mode de la console . Cette commande est disponible depuis la version 2.4. Voir BatBox (/p).

  • /n : Retourne à la ligne, tout en restant en dessous des dernières coordonées ou un goto a eu lieu. Disponible depuis la version 3.0 . Voir BatBox (/n).

  • /x sprite : Affiche le fichier sprite si sprite est un sprite généré avec la commande editor.exe du toolkit BG http://consolesoft.com/. Voir BatBox (/x).

Si vous voulez créer des graphismes en utilisant BatBox, il est possible de générer une ligne de commande pour BatBox à partir d'un fichier texte contenant une image en ASCII-Art, en utilisant l'utilitaire SpriteBox.

Notes

On peut combiner plusieurs fonctions sur une seule ligne de la commande BatBox, comme le signalent les ....

Ainsi, le script suivant:

BATBOX /g 10 10
BATBOX /d "Hello world"
BATBOX /g 0 0

Peut être raccourci en :

BATBOX /g 10 10 /d "Hello world" /g 0 0

Il est possible de spécifier les nombres dans BatBox dans trois bases différentes, et cela pour toutes les options. La première est la base décimale, pour spécifier cette base, il suffit de mettre le nombre en décimal. La seconde est la base hexadécimale, particulièrement utilisée pour les couleurs. Pour spécifier un nombre dans la base hexadécimale, il faut ajouter à ce nombre le préfixe 0x. La dernière est la base octale, dont l'usage est déconseillé car cette base est peu usitée. Pour spécifier un nombre en base octale, il faut ajouter à ce nombre le préfixe 0.

BatBox étant écrit en langage d'assemblage, il n'est pas forcément aisé de le porter vers un autre système d'exploitation, c'est pourquoi il n'est disponible que sous MS-WINDOWS.
