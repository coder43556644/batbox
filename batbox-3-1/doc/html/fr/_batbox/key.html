BatBox (/k)

Récupère un appui sur une touche, soit en attendant que l'utilisateur appuie sur une touche, soit sans l'attendre. Et retourne son code dans %ERRORLEVEL%.

Synopsis

BATBOX /k
BATBOX /k_

Attend ou vérifie l'appui sur une touche.

  • /k : Spécifie que BatBox doit attendre jusqu'à ce que l'utilisateur presse une touche

  • /k_ : Spécifie que BatBox ne doit pas attendre l'appui sur une touche. BatBox retourne alors le code de la touche presséé ou 0

Notes

Les codes des touches normales (i.e. les touche de caractères normaux) sont directement liées à leur code ASCII.

Les codes supérieurs a 255 sont des codes de touches étendues (les flèches par exemple).

On peut facilement trouver le code de la touche via ce petit script:

@ECHO OFF
:Start
BATBOX /k
ECHO code touche : %ERRORLEVEL%
GOTO :Start
